>[!def] 
>Es sei $V$ ein $K$-Vektorraum. Ein System $(\mathcal{v}_{1}, \dots, \mathcal{v}_{m} )$ von Vektoren aus $V$ heißt ***linear unabhängig***, wenn die Implikation $$c_{1} \mathcal{v}_{1} + \dots + c_{m} \mathcal{v}_{m} = \mathcal{0} \implies c_{1} = \dots = c_{m} = 0$$für alle Skalare $c_{1}, \dots, c_{m} \in K$ gilt. 
>Andernfalls heißt das System ***linear Abhängig***. 
>___
>Allgemeiner definieren wir ein beliebig indiziertes System $(v_{i})_{i \in I}$ von Vektoren aus $V$ als linear unabhängig, wenn das endliche System $\mathcal{v_{i_{1}}}, \dots, \mathcal{v_{i_{m}}}$ für jede Wahl von endlich vielen verschiedenen indizes $i_{1}, \dots, i_{m} \in I$ linear unabhängig ist.

>[!example] 15.11 Beispiel
>Diese Definition gilt genauso wie in $\mathbb{R}^n$: Lineare Unabhängigkeit bedeutet, dass *keine nicht-triviale lineare Relation* zwischen den Vektoren des Systems besteht.
>Ein Beispiel für ein unendliches linear unabhängiges System ist das System $$1, x, x^2, x^3, \dots$$aller Monome im Raum $\mathbb{R}[x]$ der Polynome, also $(\mathcal{v}_{i})_{i \in I}$ mit $\mathcal{v_{i}} = x^i, I = \mathbb{N}_{0}$.
>Die lineare Unabhängigkeit gilt in diesem Fall, weil ein Polynom $$a_{0} + a_{1}x + a_{2}x^2 * \dots + a_{n}x^n$$nur dann der Nullvektor in $\mathbb{R}[x]$ ist, also die Nullfunktion, wenn die Koeffizienten $a_{0}, \dots, a_{n}$ alle gleich $0$ sind. 
>Das gilt für Polynome von jedem Grad und damit für jede Auswahl einer Linearkombination von endlich vielen verschiedenen Monomen.

>[!TIP] **15.13** Satz
>Es sei $(\mathcal{v}_{i})_{i \in I}$ ein System von Vektoren in $V$.
>1. Genau dann ist das System **lineare abhängig**, wenn es einen Index k \in I gibt mit $$Lin(\mathcal v _{i} \ | \ i \in L) = Lin(\mathcal{v}_{i} \ | \ i \in I, i \ne k).$$
>2. Ist das System **linear unabhängig** und ist $\mathcal{v} \in V$ ein Vektor, der nicht in $Lin(\mathcal{v}_{i} \  | \ i \in I)$ enthalten ist, dann ist auhc das verlängerte System $(\mathcal{v}_{i})_{i \in I \cup {j}}$, in dem $\mathcal{v}_{j} = \mathcal{v}$ für einen neuen Indek $j \not \in I$ angehänhgt wird, linear unabhängig.

>[!def] Definition
>Ein System in einem Vektorraum $V$ heißt **erzeugend**, wenn $V = (v_{i} \ | \ i \in I)$.
>Ein Vektorraum $V$ heißt **endlichdimensional** (oder *endlich erzeugend*), wenn er von einem endlichen System von Vektoren erzeugt wird.

>[!def] Definition
>Ein System von Vektoren aus $V$ heißt **Basis** von $V$, wenn es *linear unabhängig* und *erzeugend* ist.

___

In endlichdimensionalen Vektorräumen gelten dieselben Aussagen, die wir für lineare Unterräume in $\mathbb{R}^n$ bewiesen haben:

>[!TIP] **15.14** Satz
>Es sei $V$ ein Endlichdimensionaler Vektorraum. 
>1. **Basisauswahl:** Jedes System $v_{1}, \dots, v_{m}$ von Vektoren mit $V = Lin(v_{1}, \dots, v_{m})$ enthält eine Basis von $V$. Insb. besitzt $V$ eine endliche Basis.
>___
>2. Alle Basen von $V$ haben **dieselbe Länge**.
>___
>3. Für ein System $v_{1}, \dots, v_{n}$ von Vektoren in $V$ sind äquivalent:
> 	- Das System $v_{1}, \dots, v_{n}$ ist eine *Basis* von $V$.
> 	- Jeder Vektor hat eine eindeutige Darstellung $$v = x_{1}v_{1}+\dots+x_{n}v_{n}$$als *Linearkombination* von $v_{1}, \dots, v_{n}$.
> 	- Das System ist *erzeugend und nicht verkürzbar*, d.h. kein echtes Teilsystem von $v_{1}, \dots, v_{n}$ ist erzeugend.
> 	- Das System ist *linear unabhängig und nicht verkürzbar*, d.h. jedes längere System, in dem $v_{1}, \dots, v_{n}$ vorkommen, ist linear abhängig.
> ___
> 4. Jeder lineare Unterraum von $V$ ist **endlichdimensional**.
> ___
> 5. **Basisergänzung:** Ist $U \subset V$ ein *linearer Unterraum* und $u_{1}, \dots, u_{m}$ eine *Basis* von $U$, dann gibt es $k \in \mathbb{N}_{0}$ und Vektoren $v_{1}, \dots, v_{k} \in V$ derart, dass das System $u_{1}, \dots, u_{m}, v_{1}, \dots, v_{k}$ eine Basis von $V$ ist.
>___
>___
>Der Beweis ist analog.

Für *unendlichdimensionale Vektorräume* gelten diese Aussagen entsprechend, wobei man fü rden Beweis allerdings ein Abstraktes Prinzip der Mengenlehre, das *Zornsche Lemma* heranzieht.

___

>[!example] 15.15
>Es gibt trotzdem unendlichdimensionale Vektorräume, für die wir direkt eine Basis angeben können:
>Das System $1, x, x^2, \dots$ im Vektorraum $\mathbb{R}[x]$ ist linear unabhängig und per Definition erzeugend, also eine Basis. 

>[!def] Definition
>Die **Dimension** eines endlichdimensionalen Vektorraums $V$ ist die Länge einer Basis von $V$ und wird mit $dim(V)$ bezeichnet.

>[!TIP] **15.16** Satz ***Dimensionsformel für lineare Unterräume***
>Es sei $V$ ein endlichdimensionaler Vektorraum und seien $U_{1}$ und $U_{2}$ zwei lineare Unterräume vom V.
>1. Falls $U_{1} \subset U_{2}$, dann gilt $dim(U_{1}) \le dim(U_{2})$. Dabei ist $dim(U_{1}) = dim(U_{2}) \Leftrightarrow U_{1} = U_{2}.$
>   ___
>2. Es gilt $$dim(U_{1} + U_{2}) = dim(U_{1}) + dim(U_{2}) - dim(U_{1} \cap U_{2}).$$

>[!TIP] 15.17 Korollar
>Ist $V$ ein endlichdimensionaler Vektorraum der Dimension $n$ und ist $v_{1}, .., v_{n}$ ein System aus $n$ Vektoren in $V$, dann sind äquivalent:
>1. ... sind eine Basis von $V$.
>2. ... erzeugen $V$.
>3. ... sind linear unabhängig.

>[!example] Beispiel
>Die drei Vektoren $$\vec{e}_{1} = (1, 0, 0), \ \ \ \vec{e}_{1} + \vec{e}_{2} = (1, 1, 0), \ \ \ \vec{e}_{1} + \vec{e}_{2} + \vec{e}_{3} = (1, 1, 1)$$in $K^3$ sind lin. ua. Nach dem vorangegangenen Korollar sind sie damit eine Basis von $K^3$.
>Mit anderen Worten, es muss $Lin(\vec{e}_{1}, \vec{e}_{1} + \vec{e}_{2}, \vec{e}_{1} + \vec{e}_{2} + \vec{e}_{3}) = K^3$ gelten, ohne dass wir das noch extra nachprüfen müssen. 

Sind $V$ und $W$ zwei $K$-Vektorräume, dann ist das kartesische Produkt $$V \times W = \{ (v, w) \ | \ v \in V \text{und} w \in W \}$$wieder ein $K$-Vektorraum, wenn man Addition und Skalarmultiplikation komponentenweise definiert, also durch $$(v,w) + (v',w') = (v * v', w + w') \land a(v,w) = (av, aw)$$für $v, v' \in V, w, w' \in W$ und $a \in K$.

>[!def] Definition
>Ein Vektorraum $V \times W$ heißt das **direkte Produkt** von $V$ und $W$.
