#### Aufgabenstellung

Schreiben Sie ein Programm, das folgende Ausgabe produziert: Die Buchstaben `a,b,c,d,e,f` werden entsprechend der Ziffern Ihrer persönlichen Matrikelnummer wiederholt und hintereinander geschrieben (von links nach rechts).

Zum Beispiel:

Die Matrikelnummer lautet 123456: `abbcccddddeeeeeffffff`.

Für diese Aufgabe dürfen Sie ausschließlich die Bibliothek `cstdio` importieren. Die Verwendung von Wiederholungsanweizzsungen (insofern Ihnen bekannt) ist nicht erlaubt. Achten Sie auf die Formatierung: Die Ausgabe muss in einer einzelnen Zeile erfolgen.


#### Quellcode

``` c++
#include <cstdio>

int main() {
printf("aabbbbbbbcccccceeeeeeeeefff");
return 0;
}
```
