# Physikalische Größen
#### Messgröße
Messbare physikalische Eigenschaft eines Objektes

#### Messwert
Beschreibung der *Messgröße* durch Produkt **Zahl** x **Einheit**
- beschreibt also die **Quantität** einer physikalischen Größe, welche sich zusammensetzt aus:
	- Zahlenwert
	- Einheit 
		- *Beispiel:* Längenangabe $\begin{bmatrix} l\end{bmatrix} = \text{m},  \ \ \ l = 5,6\text{ m}$

⇒ Einheiten erlauben **Vergleichbarkeit** der Messungen physikalischer Größen
⇒ Eindeutige **Definition von EInheiten** sind essenziell
	Daher sind physikalische Größen *gesetzlich geregelt*.
	 ⤷ DE: *Physikalisch Technische Bundesanstalt*, Braunschweig

# SI-Einheiten
#### SI-System (Système International d'Unitès)
Weltweit gültiges System der Maßeinheit für physikalische Größen
 ⤷ Eingeführt *1960*

| **Basisgröße**  | **Basiseinheit** |
| --------------- | ---------------- |
| Länge           | Meter (m)        |
| Masse           | Kilogramm (kg)   |
| Zeit            | Sekunde (s)      |
| El. Stromstärle | Ampère (A)       |
| Temperatur      | Kelvin (K)       |
| Stoffmenge      | Mol (mol)        |
| Lichtstärke     | Candela (cd)     |
